# RivalrySMP
RivalrySMP is a Minecraft server that focuses on competitive gameplay, community events, And STEALING HEARTS. Join us for an exciting experience filled with challenges, alliances, and epic battles!
## Features
- Lifesteal Plugin - Gain health by defeating other players!
- Custom Events - Participate in regular events and win amazing prizes.
- And a lot of fun!

3. Join and start your adventure!
